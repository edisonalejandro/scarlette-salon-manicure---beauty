# Scarlette Salon – Manicure & Beauty

Aplicación web para el emprendimiento de manicure **Scarlette Salon**.

## Descripción

Sitio web profesional para salón de manicure y belleza, con las siguientes secciones:

- **Inicio** – Hero con llamadas a la acción
- **Nosotros** – Historia, valores y propuesta de valor
- **Servicios** – Manicure clásico, gel, acrílico, pedicure spa, nail art y tratamientos
- **Galería** – Muestra de trabajos con filtros por categoría
- **Precios** – Planes Básico, Estándar y Premium
- **Testimonios** – Reseñas de clientas
- **Contacto / Reservas** – Formulario de cita con integración a WhatsApp

## Tecnologías

- HTML5 semántico
- CSS3 (variables, Grid, Flexbox, animaciones)
- JavaScript vanilla (sin dependencias externas)
- Google Fonts (Playfair Display + Lato)

## Estructura del proyecto

```
/
├── index.html          # Página principal
├── css/
│   └── styles.css      # Hoja de estilos principal
├── js/
│   └── main.js         # Lógica JavaScript
└── images/             # Carpeta para imágenes
```

## Cómo usar

1. Clona el repositorio.
2. Abre `index.html` en cualquier navegador moderno.
3. Actualiza el número de WhatsApp en `js/main.js` (variable `waNumber`).
4. Reemplaza las imágenes placeholder con fotografías reales del salón.

## Personalización

- **Colores**: edita las variables CSS en `css/styles.css` (sección `:root`).
- **Número de WhatsApp**: reemplaza `573000000000` en `js/main.js`.
- **Redes sociales**: actualiza los enlaces en el footer de `index.html`.
- **Precios**: edita los valores directamente en `index.html`.
